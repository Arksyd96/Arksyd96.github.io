Rings

A ring is a set of elements with two binary operations that satisfies the following properties:
Let A be a set, and +:A×A→A+: A \times A \rightarrow A+:A×A→A and ×:A×A→A\times: A \times A \rightarrow A×:A×A→A be two binary operations, then :

  1. (A,+,0)(A, +, 0)(A,+,0) is an abelian group, where 0 is the neutral element. An abelian group is a commutative group (x+y=y+xx + y = y + xx+y=y+x).
  2. (x×y)×z=x×(y×z)(x \times y) \times z = x \times (y \times z)(x×y)×z=x×(y×z) for all x,y∈Ax, y \in Ax,y∈A (associativity).
  3. x×y=y×xx \times y = y \times xx×y=y×x for all x,y∈Ax, y \in Ax,y∈A (commutativity).
  4. x×(y+z)=(x×y)+(x×z)x \times (y + z) = (x \times y) + (x \times z)x×(y+z)=(x×y)+(x×z) for all x,y,z∈Ax, y, z \in Ax,y,z∈A (distributivity).
  5. ∃1∈A\exists 1 \in A∃1∈A such that x×1=1×x=xx \times 1 = 1 \times x = xx×1=1×x=x for all x∈Ax \in Ax∈A (neutral element of ×\times×).
  6. ∃0∈A\exists 0 \in A∃0∈A such that x+0=0+x=xx + 0 = 0 + x = xx+0=0+x=x for all x∈Ax \in Ax∈A (neutral element of +).

A ring is noted A=(A,+,×)A = (A, +, \times)A=(A,+,×) or A=(A,+,×,0,1)A = (A, +, \times, 0, 1)A=(A,+,×,0,1).

Examples of rings: Z\mathbb{Z}Z with addition and multiplication is a ring (Z,+,×,0,1)(\mathbb{Z}, +, \times, 0, 1)(Z,+,×,0,1).
